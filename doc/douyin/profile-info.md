# 抖音博主信息采集

## 功能目的

按抖音公开主页采集博主资料、互动统计和公开标签，不读取作品列表数据。

## 输入与运行选项

- 输入格式：`https://www.douyin.com/user/用户ID`。
- 支持多个主页和批量编辑。
- 支持 `100 - 6000 ms` 随机主页间隔与循环监控。
- 不进行抖音登录状态检测。

## 执行流程

1. 使用独立任务标签页打开目标主页。
2. 等待公开资料区连续稳定。
3. 读取头像、昵称、抖音号、互动统计、简介及可见标签。
4. 每个主页仅保存最新一条资料。

## 数据字段

`profileId`、`douyinId`、`avatar`、`nickname`、`following`、`followers`、`likes`、`ipLocation`、`age`、`location`、`bio`、`profileTags`、`profileRawText`、`profileUrl`、`capturedAt`。

其中 `avatar` 读取博主主页顶部的本人头像，不使用认证徽标图片；`profileRawText` 保留公开资料区原文。

## 数据与记录

- 运行记录支持主页和状态筛选，任务编号可打开错误与数据明细。
- 数据最多保留 3000 条，每种运行状态最多保留 200 条记录。
- 支持 JSON 和 CSV 导出。

## 注意事项

不进行登录检测不代表可以绕过平台安全验证。出现验证码或网络环境提示时，请在对应抖音标签页完成处理后再运行。

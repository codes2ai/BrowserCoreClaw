# Google 新闻监控

## 功能目的

按一个或多个关键词搜索 Google 新闻，并采集最近一小时内可见的新闻结果。适合持续追踪指定主题、品牌或事件的公开新闻变化。

## 输入与运行选项

- 关键词：支持逐条新增、删除、批量编辑，以及 JSON 参数编辑。
- 每个关键词结果数：可设置 1–100 条；结果不足时保存页面实际数量。
- 关键词间隔：设置 `100 - 1000 ms` 这类毫秒区间；每个关键词完成后随机取值等待。
- 循环监控：启用后按设定分钟数重复整轮任务，直到手动停止。

## 执行流程

1. 复用已有 Google 标签页；没有可用页面时自动打开。
2. 依次打开“Google 搜索 + 新闻模式 + 最近一小时”结果页。
3. 等待结果列表稳定后，按页面顺序读取结果。
4. 每个关键词写入独立运行记录；数据按关键词与结果链接去重保存。

## 数据字段

`keyword`、`title`、`description`、`source`、`publishedAt`、`url`、`capturedAt`。

## 数据与记录

- 运行记录可按关键词、状态筛选；每种状态最多保留 200 条。
- 数据表最多保留 3000 条，支持 JSON 与 UTF-8 BOM CSV 导出。
- 点击任务编号可查看该关键词的状态、耗时、数据量及错误信息。

## 注意事项

真实采集依赖扩展的 `tabs`、`debugger` 权限和 Google 域名权限；本地静态网页预览仅用于检查界面。
